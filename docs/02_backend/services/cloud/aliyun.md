# 阿里云

## 定位

阿里云相关服务用于承载 Research-Flow 后端的云侧基础能力，典型场景包括对象存储、日志归档与可选部署环境。

## 当前用途

- 作为对象存储候选，用于替代本地文件系统保存大体积附件
- 作为云部署候选环境，用于后续扩展后台任务或共享部署

## 当前边界

- 当前阶段以后端本地优先为主，阿里云接入不作为 `P0` 必需能力
- 若启用云存储，需要与 `physical_item.storage_type` 和 `storage_path` 一致建模
- 凭证、地域、Bucket 等敏感信息不得写入仓库

## 后续扩展

- `OSS` 接入
- 云侧日志归档
- 备份与恢复策略
